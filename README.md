# E-Posyandu
E-Posyandu adalah sistem berbasis web yang dikembangkan untuk membantu kader posyandu dalam melakukan pencatatan 
dan pengelolaan data pemeriksaan secara digital. Sistem ini mempermudah proses pencatatan data anak, pemeriksaan, serta penyimpanan riwayat

# Fitur
1. Login Admin
2. Manajemen Data Anak (Tambah, Edit, Hapus)
3. Pencatatan Pemeriksaan
4. Riwayat Pemeriksaan

# Cara Menjalankan Project
1. pindahkan project ke
   ```bash
   C:\xampp\htdocs\
   ```
   atau ke
   ```bash
   C:\laragon\www\
3. Jalankan Apache dan MySQL melalui XAMPP atau Laragon
4. Import database ke phpMyAdmin.
5. Akses melalui browser
